# Bank Customer Churn & Revenue Risk Analysis

## Project Overview
Analyzed 10,000 banking customers to identify churn drivers and revenue-risk segments using Python, SQL, and Power BI.

## Tools Used
- Python
- Pandas
- SQL (MySQL)
- Power BI

## Features
- Customer churn analysis
- Revenue risk assessment
- Customer segmentation
- Interactive dashboards
- Business recommendations

## Key Findings
- Germany showed the highest churn rate (~32%).
- Female customers churned more than male customers.
- Customers with 3–4 products had the highest churn rates.
- High-balance customers contributed most to revenue risk.

## Business Recommendations
- Target German customers with retention campaigns.
- Improve engagement for high-balance customers.
- Investigate dissatisfaction among multi-product customers.
- Launch loyalty programs for at-risk customer segments.
